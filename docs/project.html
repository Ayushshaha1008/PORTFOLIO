Project Overview

This project is a Student Registration App built using Node.js for frontend and Maven (Spring Boot) for backend. It demonstrates 3-tier architecture with database, application, and frontend layers.

Documentation / Process

  • Setup MariaDB for database storage
  • Configured Apache Tomcat and deployed student.war
  • Implemented REST APIs for CRUD operations
  • Frontend connects via Node.js to backend APIs
